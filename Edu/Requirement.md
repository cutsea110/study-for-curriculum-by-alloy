# 概要

学士号の取得や教職資格などの資格を取得するという修学の目的を要件という概念で定義する.

## 要件

現状の教務システムでは,卒業要件(あるいは卒業進級要件)は卒業と同時に学士号の取得を含む要件と位置づける.
卒業要件以外の要件はその他の資格要件として定義しており,教職要件は資格要件の部分集合という位置づけとする.

```alloy
abstract sig 要件 {
}
one sig 卒業要件 extends 要件{
}
sig 資格要件 extends 要件{
}
sig 教職要件 extends 資格要件{
}
```
